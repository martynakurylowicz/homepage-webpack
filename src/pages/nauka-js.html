JS - JavaScript

Programowanie

Szkolenie obejmuje wersję ECMAScript2015+ czyli ES6+

Z czego składa się JS:

  • Wartości -
  • Operacje porównywania
  • Literały
  • Zmienne (let, const, var) / nazwy camelCase
  • Obiekty
  • Funkcje

linkowanie skryptu <script src="main.js"></script>

; - średnik oznacza linię skończoną

Zmienne: const - stałe połączenie z jedną wartością / let - połączenie które można przepinać, ale zdefiniowane raz

Typy proste:

  • string - tekst / otaczamy ' lub "
  • number - liczba
  • bigint - duża liczba / większa niż 2^53
  • boolean - true lub false
  • null - wartość celowo niezidentyfikowana
  • undefined - brak definicji
  • *symbol - tworzy unikalne opakowanie

Poznane funkcje:

  • alert('treść'); - komunikator na początku / okienko
  • console.log('treść'); - przy badaniu strony w odzielnym okienku wyświetla się jako komunikat/wiadomość dla developerów
  • console.log(name); lub console.log(age); - funkcja ze zmienną / zapis bez apostrofów
  • const name='Maciek'; - stała zmienna - przypisanie wartości zmiennej / przy dłużej nazwie zmiennej należy użyć camelCase
  • const age=32; - stała zmienna - przypisanie wartości zmiennej / przy dłużej nazwie zmiennej należy użyć camelCase
  • ` - backtick - specjalny ciąg znaków zwany literałem
  • console.log(`Nazywam się ${name} i mam ${age} lat`) - ${zmienna} - w użyciu mieszanych ciągów znaków w backticku
  • typeof nazwaZmiennej - sprawdza jakiego typu jest zmienna
  • document.querySelector('.selector') - podpinanie się pod elementy html / selektor taki sam jak w CSS / warto dodać w BEM modyfikator js czyli np. .header__menu--js
  • function.calculate(myNumber) {return myNumber*7;} - funkcja w JS - służy do wielokrotnego użycia w kodzie / przy braku wywoałania opisze jako undefined / może zawierać kilka zmiennych
  • const calculate = (myNumber) => {myNumber = myNumber+3; console.log(myNumber); return myNumber*7} - (Fat) arrow function - wywołanie np. calculate(2); //35
  • const calculate = (myNumber) => (myNumber+3)*7; - (Fat) arrow function - wywołanie np. calculate(2); //35
  • deathStar.name - wypisuje właściwość obiektu - kropka wywołuje właściwość
  • import - importowanie między plikami
    import nazwaRzeczy from 'nazwa-modułu/ścieżka_do_pliku'
    import {pierwszy, drugi} from 'nazwa-modułu/ścieżka_do_pliku'
    import {bardzodługanazwa as pierwszy} from 'nazwa-modułu/ścieżka_do_pliku'
    import* as newName from ... - importowanie wszystkich elementów z pliku .js
    export - eksport funkcji z jednego pliku js do innego
    export default - eksport z dowolną zmienną
  • document.documentElement.style.setProperty('--zmienna', 'wartość') - zmiana wartości zmiennej CSS

Metoda porównania:

if('JavaScript!='Java'){//wykonaj kod jeśli true}

AND - wszystkie warunki muszą być true
if((myVariable == 4) && (otherVar !== 'test')) {...}

OR - przynajmniej jeden warunek musi być true
if((myVariable == 4) || (otherVar !== 'test')) {...}

NOT - odwraca wartość wyrażenia logicznego
if(!('JavaScript'=='Java')) {//ten kod się wykona}

ELSE
if((myVariable == 4)) {//wykonaj kod jeśli true}
else {//wykonaj kod jeśli false}
lub np. else if(myVariable == 8) {//wykonaj kod jeśli myVariable==8}

switch (myVariable){case 4: console.log('hi 4');
break;
case 8: console.log('hi 8');
break;
default: console.log ('default response');}

Operatory porównania:

  • != - nie równa się
  • > - lewa większa od prawej
  • < - prawa większa lewej
  • >= , <= - wiadomo :)
  • == - obie strony równe
  • === - obie strony równe (sprawdza typeof)
  • !== - obie strony różne (sprawdza typeof)
